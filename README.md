# NLP Assignment Overview

This assignment covers sentiment analysis, greedy algorithm, Viterbi algorithm, and implementations of LSTM, RNN, and GRU.

## Topics

- **Sentiment Analysis**: Text sentiment classification.
- **Greedy Algorithm**: Optimization problems.
- **Viterbi Algorithm**: Hidden state sequence prediction.
- **LSTM, RNN, GRU**: Sequential data processing.

## Tools

- **Python**
- **NLTK**
- **Pytorch**

## How to Run

1. Install required packages.
2. Navigate to the project directory.
3. Run Python scripts for each topic.
